# godot-shield-mechanic
Hey! 👋 This is a small Godot 4 prototype I put together to test out an interactive defense mechanic.

The core idea is simple: bullets are falling from the sky, and you need to protect the character. Instead of moving the player, you use your mouse to grab, drag, and drop a shield to block the incoming hits.

I built this mostly to get the hang of Godot's physics, mouse inputs, and node signals. It's a tiny project, but the core loop works great. Feel free to download it, mess around with the code, and break things!
